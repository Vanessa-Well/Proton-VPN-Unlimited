## 📖 What is Proton VPN

**Proton VPN** is a Swiss VPN service created by a team of scientists from CERN who previously founded [Proton Mail](https://proton.me/mail) — the world's most popular encrypted email service. The company is headquartered in **Geneva, Switzerland**, providing users with the protection of one of the strictest data privacy jurisdictions in the world.

Proton VPN encrypts internet traffic, hides the user's IP address, and protects data from surveillance, censorship, and cyber threats. Unlike many competitors, Proton VPN operates on **open-source** principles and undergoes regular independent security audits.

### 🔒 Security Technologies

#### VPN Accelerator
Proton's proprietary technology that increases VPN connection speed by **up to 400%** through parallel data processing and protocol optimization.

#### NetShield (Ad & Malware Blocker)
A DNS-level filter that protects against:
- 🚫 Ads and trackers
- 🦠 Malware and phishing
- 📡 Telemetry and spy scripts

#### Secure Core (Double VPN)
A unique architecture where traffic passes through **two independent VPN nodes** in countries with high data protection standards (Switzerland, Iceland, Sweden). Even if the exit server is compromised — the real IP address remains hidden.

```
Your device → [Secure Core server] → [VPN server] → Internet
```

#### Kill Switch
Automatically blocks all internet traffic if the VPN connection drops, preventing accidental data leaks.

#### Split Tunneling
Lets you choose which apps and websites go through the VPN and which use the regular connection. Flexible configuration for any use case.

### 🧅 Tor over VPN

Premium users get direct access to **onion servers** that automatically route traffic through the Tor network — no need to install Tor Browser.

```
Your device → [VPN] → [Tor Network] → .onion / Internet
```

## 🚀 Quick Start

```bash
# Install Proton VPN CLI on Linux
wget https://repo.protonvpn.com/debian/dists/stable/main/binary-all/protonvpn-stable-release_1.0.3_all.deb
sudo dpkg -i protonvpn-stable-release_1.0.3_all.deb
sudo apt update && sudo apt install proton-vpn-gnome-desktop

# Connect to the fastest server
protonvpn-cli connect --fastest

# Connect to a specific country
protonvpn-cli connect --cc CH

# Connect via Secure Core
protonvpn-cli connect --sc

# Disconnect
protonvpn-cli disconnect
```
